# Kritika (Eira) — Portfolio

Personal portfolio site for **Kritika**, a.k.a. **eira** — Full-Stack Developer, Content Creator & Social Media Strategist.

🔗 **Live site:** [https://eiracodes.github.io/Portfolio-2026/)

## About

A single-page, self-contained portfolio covering:

- Profile & quick stats
- Work experience & education timeline
- Skills — development + content/marketing
- Featured projects (linked to live demos & source)
- Contact

## Tech

Plain **HTML, CSS & JavaScript** — no build step, no dependencies. One `index.html` file, ready to deploy on GitHub Pages.

## Run locally

```bash
git clone https://github.com/eiracodes/eiracodes.github.io.git
cd eiracodes.github.io
open index.html
```

## Deploy

Push to the `main` branch — GitHub Pages serves `index.html` automatically.
